# 📁 Data Description

## 📊 Dataset

This project uses a **Bank Customer Churn dataset** containing customer demographic and account information.

---

## 📌 Key Fields

* Customer ID
* Country
* Gender
* Age
* Tenure
* Balance
* Number of Products
* Credit Card Ownership
* Active Member Status
* Estimated Salary
* Churn (Target Variable)

---

## 🎯 Target Variable

* **Churn = 1** → Customer has left the bank
* **Churn = 0** → Customer is retained

---

## ⚠️ Note

The dataset is not included in this repository due to size limitations.
It can be obtained from public sources such as Kaggle.
https://www.kaggle.com/datasets/gauravtopre/bank-customer-churn-dataset
