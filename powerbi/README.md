# 📊 Power BI Dashboard

## 🎯 Purpose

To visualize churn behavior and identify high-risk customer segments.

---

## 📊 Dashboard Features

* KPI Cards:

  * Total Customers
  * Churned Customers
  * Churn Rate
* Churn Rate by Country
* Churn Rate by Customer Activity
* Churn Rate by Number of Products
* High-Risk Customer Segment Analysis
* Customer Distribution by Product Usage

---

## 🧠 Key Insight

Inactive customers with only one product represent the highest churn risk segment.

---

## 🛠 Measures Used (DAX)

* Total Customers
* Churned Customers
* Churn Rate

---

## 📌 Outcome

An interactive dashboard that supports data-driven decision-making for customer retention strategies.
